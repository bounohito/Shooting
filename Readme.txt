【ゲーム紹介】
2Dゲームだ。棒人間が巨大な球を出してジャムを蹴散らす。
ジャムはうじゃうじゃ湧いてくるから、どんどん球を出そう。
ちなみに球が画面から出るまで次の球を出せないから注意だ。
あと、出したい向きに歩かないと、その向きに球を出せないから、ジャムに近づきすぎないように注意だ。
あと、画面左右ど真ん中を通るとジャムが湧く側が変わるから、さらにそれも注意だ。
画面内のジャムを全滅させると100点を得て、4秒だけフィーバータイムになる。
フィーバータイム中はジャムははちみつになる。
倒したときに得る点は、ジャム:1点、はちみつ:8点だ。
ジャムまたははちみつに触れるとゲームオーバーだ。

【操作】WASDまたは上下左右キーで移動、スペースキーで最後に歩いた向きに球を出す。ウィンドウ右上の×か、ESCキーで終了。
